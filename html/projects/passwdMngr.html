Password Manager

Get source code

Do you have too many passwords and can't remember them? Well a password manager can help you with that. Using an SQL database, you can build a simple password manager. I implemented a simple Password Manager in C++ as well as in Python which uses a database to store the passwords. The application provides various options for the user to add/remove data.

The advantage of using a database over a text file is that updation, deletion and retrieval is very quick and easy.
